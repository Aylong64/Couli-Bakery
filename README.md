# COULi Bakery
 
A multi-page website for COULi Bakery - an online bakery in Lagos making intercontinental pastries with locally sourced Nigerian ingredients.
 Built this as a frontend project. The bakery is real, the brief was simple: give them a clean web presence that actually works on mobile and doesn't look like a template.
 
---
 
## Pages
 
- **Home** — hero section, short about teaser, featured product preview
- **About** — the bakery's story and what makes them different
- **Products** — full catalogue with prices
- **Menu** — interactive tab switcher by category (Cakes, Donuts, Cupcakes, Cookies)
- **Contact** — social links, phone, email and a review form
## Built With
 
- HTML5
- CSS3 (Flexbox, Grid, responsive media queries)
- JavaScript / jQuery (mobile nav, menu tab switching)
- Font Awesome for icons
- Google Fonts (Montserrat)

## Local Setup
 
No build tools, no dependencies. Just clone and open `index.html` in a browser.
 
```bash
git clone https://github.com/Aylong64/Couli-Bakery.git
cd Couli-Bakery
```
 
Then open `index.html` directly or use the VSCode Live Server extension.
